# Bank Marketing Classifier 📊

A machine learning pipeline that uses a Decision Tree Classifier to predict whether a customer will subscribe to a bank term deposit based on demographic and behavioral data.

## 🚀 Key Features
* Cleans and encodes categorical data using pandas
* Handles class imbalance with balanced tree weights
* Generates clear visual feature importance charts

## 🛠️ Tech Stack
* Python 3.11
* Scikit-Learn
* Pandas & NumPy
* Matplotlib

## 📦 Installation & Setup
1. Clone the repo: `git clone https://github.com/username/bank-marketing-classifier.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the model: `python model.py`
